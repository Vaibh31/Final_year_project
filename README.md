## 📖 Overview

Brainify AI is an intelligent AI-powered learning platform designed to transform the traditional learning experience into an interactive, personalized, and engaging process. The platform enables users to generate quizzes automatically, upload study materials, interact with an AI assistant, monitor academic performance, and track learning progress through visual analytics.

The system combines modern web technologies with artificial intelligence to provide a smarter way of learning and self-assessment.

---

## 🎯 Objectives

- Automate quiz generation from study materials.
- Improve student engagement through interactive learning.
- Provide personalized performance analytics.
- Enable effective self-assessment and revision.
- Create a centralized platform for learning and evaluation.

---

## ❗ Problem Statement

Students often struggle with effective revision and self-assessment due to the lack of personalized and interactive learning tools. Traditional quiz preparation requires significant manual effort and time.

Brainify AI addresses this problem by automatically generating quizzes from topics and uploaded notes while providing detailed analytics to help learners improve their performance.

## 🔧 Project Modules

### Authentication Module
Handles user registration, login, logout, and secure session management.

### Quiz Generation Module
Generates quizzes automatically based on user-selected topics and uploaded study materials.

### Notes Upload Module
Processes uploaded PDF and DOCX files, extracts content, and generates quiz questions.

### AI Chat Module
Provides an interactive chatbot to assist users with learning-related questions and guidance.

### Dashboard Module
Displays performance analytics such as quiz scores, accuracy, trends, and learning statistics.

### Leaderboard Module
Ranks users according to their performance and promotes healthy competition.

---

## 💻 Working of the System

1. Users create an account and log in.
2. Users can either choose a topic or upload study materials.
3. The system processes the content and generates quiz questions.
4. Users attempt quizzes and receive instant results.
5. Performance data is stored and analyzed.
6. Dashboards visualize learning progress and performance trends.
7. Leaderboards display rankings based on user scores.
